# SAGARMATHA-MVP
This repository contains the Minimum Viable Product (MVP) for the Sagarmatha To-Do-List, developed for the Collaborative Development module (Sprint 1).
The application is a full-stack web-based CRUD (Create, Read, Update, Delete) tool designed to help teams manage tasks efficiently. Built using a modular architecture, the system integrates HTML5 for structure, CSS3 for responsive design, and Vanilla PHP for server-side logic and MySQL database connectivity.

Core Features (Sprint 1):
Dynamic Task Creation: Users can add tasks to a centralized list via a PHP-processed form.
Real-time Data Persistence: Integration with a MySQL database to ensure data is saved across sessions.
Interactive Deletion: JavaScript-enabled confirmation dialogs to prevent accidental data loss.
Modular Backend: Clean separation of database configuration (db_config.php) and application logic (process.php).

Tech Stack: > Frontend: HTML5, CSS3, JavaScript
Backend: PHP (Vanilla)
Database: MySQL
Version Control: GitHub (Git)
